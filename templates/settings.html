About AlgoGuard AI

AlgoGuard AI is an intelligent Algorand blockchain monitoring dashboard that tracks your wallet balance, ALGO price, and market metrics in real-time.

Features

  • Real-time Algorand account monitoring
  • Live ALGO price tracking (USD)
  • Market cap monitoring
  • AI-powered risk detection and alerts
  • Historical data visualization with charts
  • Multi-user support with account isolation

Data Sources

  • Blockchain: Algorand Testnet/Mainnet via AlgoNode
  • Price Data: Binance API
  • Market Data: CoinGecko API

Security

  • Passwords are hashed and securely stored
  • Each user can only access their own data
  • No private keys or sensitive data stored
  • Read-only blockchain access
