ℹ️ About AlgoGuard AI

AlgoGuard AI is an intelligent Algorand blockchain monitoring dashboard that provides real-time insights into your wallet and the Algorand ecosystem.

Data Sources
• Blockchain: Algorand Testnet/Mainnet via AlgoNode
• Price Data: Binance API
• Market Data: CoinGecko API
Security
• Passwords encrypted with bcrypt
• User data isolation
• No sensitive data stored
• Read-only blockchain access
